# FinReason Bench

A benchmark and framework for retrieval-grounded financial reasoning systems.

## Why this project

Financial reasoning is a high-stakes task. Fluent answers are not enough if they are unsupported, inconsistent, or unsafe in risk-sensitive workflows.

This project focuses on retrieval-grounded reasoning, structured outputs, and explicit evaluation for finance-related AI systems.

## What this repo includes

- Task cases for financial reasoning
- A retrieval pipeline for evidence grounding
- Structured reasoning outputs
- An evaluation harness for answer quality and constraint checks

## Core problems

- Unsupported or hallucinated financial claims
- Missing evidence and citations
- Weak logical consistency in generated answers
- Risk-unsafe recommendations in high-stakes settings
