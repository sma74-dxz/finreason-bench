# Problem Definition

## Background
Financial AI systems often produce fluent answers, but fluency does not guarantee correctness, evidence support, or risk awareness.

## Why this matters
In high-stakes financial workflows, unsupported claims can lead to poor decisions and loss of trust.

## Key failure modes
- Hallucinated facts
- Unsupported claims
- Missing citations
- Logical inconsistency
- Risk-unsafe recommendations

## Project direction
This project explores retrieval-grounded financial reasoning with explicit evaluation and structured outputs.
